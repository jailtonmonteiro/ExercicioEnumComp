## Sobre o Projeto

Projeto desenvolvido para explicação de assuntos como composição e enumeração dentro da liguagem C#, no curso de orientação a objetos com C#.

## Estrutura de aplicação

/Entities      ## Pasta destinada a classes da aplicação
  /Enums       ## Pasta dedicada a arquivos de enumerações

## Conceitos abordados

* CLASSES
* ATRIBUTOS
* METODOS
* MODIFICADORES DE ACESSO
* CONTRUTORES
* ENCAPSULAMENTO
* ESTRUTURAS DE REPETIÇÃO
* LISTAS
* CONDICIONAIS
* ENUMERAÇÃO
* COMPOSIÇÃO
